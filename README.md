なうクエークは、地震速報をリアルタイムに受信し、今何をすべきなのかをタイムライン把握できるようにする「**減災支援アプリ**」です。

## 特徴
- [] **リアルタイム地震速報**: 地震発生と同時に速報を受信し、即座に通知します。
- [] **タイムライン**: 事前に設定しておいたタイムラインに沿って、地震発生後の行動をガイドします。
- [] **カスタマイズ可能なタイムライン**: ユーザーは自分の生活スタイルや地域の特性に合わせて、タイムラインを自由に編集できます。
- [] **オフライン対応**: インターネット接続がなくても、地震発生後の行動ガイドを利用できます。
